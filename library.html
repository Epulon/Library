JavaScript


First section for the most part follows SoloLearn's tutorial on JavaScript


Basics


  • Javascript is inserted in HTML document between <script> and </script> tags.


  • To get Javascript to print output within HTML document:

    • document.write("Something, something, complete.")


  • For web console output function console.log() is used. Example:

    • console.log("Hello console")


  • Variables are containers for storing data.
    • For example assigning value 10 to variable x

      • var x = 10;

    • For further example see console/code ex1.
    • Some naming rules for variables:
      • JavaScript variables are case sensitive. Var x and var X are different variables
      • First char. has to be a letter, underscore (_) or dollar ($)
      • No spaces
      • No math operators (* / + -)
      • No special char (# %)

  • Comments in JavaScript are placed between /* and */ or after // for single line comments.

  • Data types:
    • Numbers: var num = 12; or var price = 42.68;

  • Strings are basically text wrapped in quotes:

    • var = name "Mirsad";


  • Backslash escape character used for putting quotes or other special characters within a string:
    • class="inscode"> var sayHello = 'Hello, I am a \"programmer\"!'
    • Some other uses:
      • \' - Single quote
      • \" - Double quote
      • \\ - Backslash
      • \n - New line
      • \r - Carriage return
      • \t - Tab
      • \b - Backspace
      • \f - Form feed

  • Boolean data types are usefull when there are only two possible values, 1 or 0, true or false....

    • var isActive = true;
    var isChristmas = false;


  • Arithmetic operators, JavaScript uses standard arithmetic operators:
    • + Addition
    • - Substraction
    • * Multiplication
    • / Divison
    • % Modulus (remainder)
    • ++ Increment (ie. var a=10, a++ -> a=11)
    • -- Decrement (ie. var a=10, a-- -> a=9

  • Assignment Operators are used to assign values to JavaScript variables.

    • Operator Example Is equvalent to:
    = x = y x = y
    += x += y x=x+y
    -= x -= y x=x-y
    *= x *= y x=x*y
    /= x /= y x=x/y
    %= x %= y x=x%y

  • Comparison Operators are used to check if variables or values are different
    -Response can be either true or false.
    -They only work when comparing same data types (numbers with numbers, strings with strings...)

    • var num = 10;

    num == 8;

    Returns-false
