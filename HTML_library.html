Inline and Block elements


  • Block level elements start from a new line (h1, form, li, ol, ul, p, pre, table, div...)

  • Inline elements are displayed withouth line breaks(b, a, img, input, span, em...)

  • The div element is a block-level element that is often used as a container for other HTML elements.

  • The Span element defines an inline section of the document (part of line or div)

  • Other elements:
    • IFRAME-Inline frame
    • INS-Inserted text
    • MAP-Image map
    • APPLET-Java applet
    • OBJECT-Embeded object
    • SCRIPT-Script within HTML document

Forms


  • Forms (form) are used to collect information from the user
  • Action (action) attribute is used to point to the webpage that will load after the user submits the form
  • GET-data will be visible on page address
  • POST-data not visible in page address
  • Input types:
    • text-text box
    • password-password box
    • radio-single choice prompt
    • checkbox-multiple choice prompt
    • Submit button submits form into its action attribute
